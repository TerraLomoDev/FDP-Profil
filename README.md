# FDP-Profil
Patrick van Rossum FDP Profil

## Entwicklung

```bash
npm install
npm run dev
```

Die Startseite liegt in `src/App.jsx`, das Styling in `src/styles.css`.

## GitHub Pages

Diese React/Vite-Seite muss gebaut werden, bevor GitHub Pages sie ausliefern kann.
Das Deployment laeuft ueber `.github/workflows/deploy.yml`.

In GitHub unter `Settings -> Pages` als Source `GitHub Actions` auswaehlen.
